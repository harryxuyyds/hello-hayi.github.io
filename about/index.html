关于此站点

如你所见，这个站点的载入速度比较一般。一方面确实是几个图片数据拖慢了很大一部分进度，更主要的可能还是 Github Pages 的锅（非常的无奈呜呜呜）。所以啦，真的非常非常感谢有耐心等这个加载过程完成的你们！

聊聊之后对站点建设的计划吧。可能短期内会去选购一个阿里云的域名，然后横评几个云服务器。最后应该大概率会把站点建在阿里云的服务器上。

说起来搭建这个站点的历程，确实是挺不容易的哈哈。我超级怀疑第一次搭建失败是因为那天上午折腾了一会儿 Electron 的问题，第二次重装了 Node.js 就都正常了。

对啦，自己还有点强迫症哈哈，所以会把之前写的 Markdown 也都归档到站点里去。
